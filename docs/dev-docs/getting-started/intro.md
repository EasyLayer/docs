## Open-source & Contribution

EasyLayer is an open-source project (see [LICENSE](https://github.com/easylayer/base/blob/master/LICENSE) TODO file for more information). The core project, as well as the documentation and any related tool can be found in the [EasyLayer](https://github.com/easylayer) GitHub organization.

As it goes hand in hand with the open-source ecosystem, EasyLayer is open to contributions. The EasyLayer team appreciates every contribution, be it a feature request, bug report, or pull request. The following GitHub repositories are open-source and contributions-friendly:

- [`easylayer/base`](https://github.com/easylayer/base): main repository of EasyLayer, which contains the base solution of the our APIs. You can find the core plugins, and the whole code that runs your EasyLayer applications. Please read the [`CONTRIBUTING.md`](https://github.com/easylayer/base/blob/master/CONTRIBUTING.md) file to have more information about contributions to the main repository.
- [`easylayer/docs`](https://github.com/easylayer/docs): contains the whole documentation of EasyLayer.

## EasyLayer Community

EasyLayer is a community-oriented project with an emphasis on transparency. The EasyLayer team has at heart to share their vision and build the future of EasyLayer with the EasyLayer community. This is why all insights are very important and will help steer the project in the right direction, any community member is most welcome to share ideas and opinions there.

Want to join the community?
You can join [GitHub](https://github.com/easylayer), the [Forum](https://github.com/EasyLayer/base/discussions) to share your ideas and opinions with other community members and members of the EasyLayer team. If you're looking for news and updates about EasyLayer, [Twitter](https://twitter.com/EasyLayerWeb3) and the [blog](https://easylayer.io/blog) are pretty good places to start!

## Support

EasyLayer is offered as free and open-source for users who wish to self-host the software. When having an issue or a question, the [forum](https://github.com/EasyLayer/base/discussions) is great first place to reach out for help. Both the EasyLayer community and core developers often check this platform and answer posts.
